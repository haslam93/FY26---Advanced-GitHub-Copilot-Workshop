## [1.0.0] — 2026-02-23

### Added
- Initial repository structure with 10 modules
- `.github/copilot-instructions.md` — live repo-level C#/.NET instructions demo
- `.github/AGENTS.md` — multi-agent compatible always-on instructions
- `.github/prompts/` — 5 reusable prompt files: `code-review`, `generate-tests`, `explain-legacy`, `sql-query`, `modernize-dotnet`
- **Module 01 — Customization**: instruction file types, repo vs workstation scope, prompt files, agent modes, agent skills, org-level instructions
- **Module 02 — VS Code Agents**: Background Agent, Cloud Agent, sub-agents, agent selection guide
- **Module 03 — MCP Servers**: Azure DevOps MCP, Azure MCP, GitHub Remote MCP, Playwright MCP configs and usage examples
- **Module 04 — Copilot CLI**: `gh copilot suggest/explain`, shell aliases, Copilot Fleet
- **Module 05 — App Modernization**: .NET Upgrade Assistant, Java Modernization Assistant, legacy code samples
- **Module 06 — QA & Testing**: xUnit + Moq, test plan generation, Playwright UI tests, accessibility testing
- **Module 07 — Databases**: mssql extension, SQL prompting patterns, sample permits schema, stored procedures
- **Module 08 — Models & Context**: full model reference table with premium multipliers (sourced 2026-02-23), context management guide
- **Module 09 — Copilot on GitHub.com**: Coding Agent, Code Review in PRs, Copilot Spaces
- **Module 10 — Hands-on Lab**: 5 self-contained exercises (~90 min total)
